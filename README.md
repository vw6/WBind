# WBind

The project is written in powershell and c #, packed with ps2exe utility.
You can add unlimited number of presets.

the file should look like this ...

```
{
    "settings": [
      {
        "Debug": "yes"
      }
    ],
    "textAndKey": [
      {
        "name":"Сказать привет",
        "key": "F4",
        "text": "Привет"
      }
    ]
  }
```
